**Create `lib/queries/briefing.ts`** with `import "server-only"`:

Export `fetchBriefingData(accountId: string)` that returns a `BriefingData` object:
- Compute two 7-day windows: current (today minus 7 days to today) and prior (today minus 14 days to today minus 7 days)
- Use `Promise.all` to fetch data in parallel across all 4 formats for BOTH periods:
  - RSA: use `fetchRsaCreatives` (import from rsa.ts) for current and prior
  - PMax: use `fetchPmaxAssetGroups` (import from pmax.ts) for current and prior
  - Display: use `fetchDisplayCreatives` (import from display.ts) for current and prior
  - Video: use `fetchVideoCreatives` (import from video.ts) for current and prior
- That's 8 parallel queries (4 formats x 2 periods)
- Return `BriefingData`:
  ```typescript
  interface BriefingData {
    currentPeriod: { from: string; to: string };
    priorPeriod: { from: string; to: string };
    rsa: { current: CreativeAgg[]; prior: CreativeAgg[] };
    pmax: { current: CreativeAgg[]; prior: CreativeAgg[] };
    display: { current: CreativeAgg[]; prior: CreativeAgg[] };
    video: { current: CreativeAgg[]; prior: CreativeAgg[] };
  }
  ```
  where `CreativeAgg` is a minimal interface: `{ adId: string; kpiValue: number; impressions: number; label?: string }`
- Compute kpiValue based on account's primaryKpi setting
- For RSA label, use joined headline text. For PMax, use assetGroupName. For Display/Video, use adName.

**Create `components/briefing/ChangeIndicator.tsx`** -- "use client" component:
- Props: `value: number` (change percentage), `kpiType: 'cpa' | 'roas'`, `size?: 'sm' | 'md'`
- Render an up/down arrow with colour-coded percentage
- For CPA: positive change = red (bad, cost went up), negative = green (good)
- For ROAS: positive change = green (good, return went up), negative = red (bad)
- Use lucide-react `TrendingUp` and `TrendingDown` icons
- Format: "+23.4%" or "-12.1%" with appropriate colour

**Create `components/briefing/BriefingSection.tsx`** -- "use client" component:
- Props: `title: string`, `icon: React.ElementType`, `children: React.ReactNode`, `count?: number`
- Renders a card section with:
  - Header with icon, title, and optional count badge
  - White background with border (design system surface-gridline)
  - Children slot for section content
