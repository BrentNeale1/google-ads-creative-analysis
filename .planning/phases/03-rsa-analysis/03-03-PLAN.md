Create `lib/queries/rsa.ts` with `import 'server-only'` guard. Follow the exact pattern from `lib/queries/dashboard.ts` (Drizzle ORM, conditional where clauses, sum aggregations).

**Functions to implement:**

1. `fetchRsaCreatives(accountId, dateFrom, dateTo, campaignId?, adGroupId?) -> RsaCreativeAggregated[]`
   - Query `rsaDaily` grouped by adId, campaignName, adGroupName, headlines, descriptions, adStrength
   - Sum: impressions, clicks, costMicros, conversions, conversionsValue
   - Apply optional campaign/adGroup filters
   - Return with derived fields pre-computed: `ctr` (clicks/impressions), `cvr` (conversions/clicks), `cpa` (costAud/conversions), `roas` (conversionsValue/costAud)
   - Guard all divisions against zero denominators (return 0)
   - Use `convertMicrosToAud` from `@/lib/constants/formatting` for cost conversion
   - Type the return with a clear interface including all raw + derived metrics

2. `fetchRsaAssets(accountId, dateFrom, dateTo, adId?) -> RsaAssetAggregated[]`
   - Query `rsaAssetDaily` grouped by adId, assetResource, fieldType, textContent, performanceLabel
   - Sum: impressions, clicks, costMicros, conversions
   - Compute derived: ctr, cpa
   - Filter by optional adId for drill-down
   - Return textContent alongside metrics so the UI can show "Get 50% Off Today" not "customers/123/assets/456"

3. `fetchRsaCombinations(accountId, dateFrom, dateTo, adId?) -> RsaCombinationAggregated[]`
   - Query `rsaCombinationDaily` grouped by adId, headlines, descriptions
   - Sum: impressions only (no clicks/conversions -- Google limitation)
   - Do NOT compute CTR or CPA -- these metrics don't exist for combinations
   - Sort by impressions descending (most-served combinations first)

4. `fetchRsaPortfolioAvg(accountId, dateFrom, dateTo) -> PortfolioAvg`
   - Compute portfolio-wide averages needed for underperformer diagnosis
   - Average impressions per creative, average CTR, average CVR, average CPA
   - This is a single aggregation query across rsaDaily for the date range
   - Returns: `{ avgImpressions, avgCtr, avgCvr, avgCpa }`

Export all interfaces alongside functions. Use named exports per CLAUDE.md code style.
