  <action>
Create `components/dashboard/FilterBar.tsx` as a "use client" component.

**Props:**
- `campaigns`: `Array<{ id: string, name: string }>` -- available campaigns for the selected account
- `adGroups`: `Array<{ id: string, name: string, campaignId: string }>` -- all ad groups, filtered client-side by selected campaign

**URL State (via nuqs):**
Use `useQueryStates` from `nuqs` with the same parsers as `searchParams.ts`:
- `range`: `parseAsStringLiteral(['7d', '30d', '90d', 'custom'] as const).withDefault('30d')`
- `from`: `parseAsIsoDate`
- `to`: `parseAsIsoDate`
- `campaign`: `parseAsString`
- `adGroup`: `parseAsString`

**Layout:**
Horizontal bar with `flex items-center gap-3 flex-wrap` styling. Background white, border-b border-surface-gridline, px-6 py-3.

**Elements (left to right):**

1. **Date range presets:** Three buttons (7d, 30d, 90d) styled as pill buttons. Active preset gets `bg-brand-blue text-white`, inactive gets `bg-surface-background text-gray-600 hover:bg-surface-gridline`. On click: `setFilters({ range: preset, from: null, to: null })`.

2. **Custom date range:** A "Custom" button that toggles a small inline form showing two `<input type="date">` fields (From, To). When custom is active, show the date inputs. On date change: `setFilters({ range: 'custom', from: new Date(value), to: new Date(value) })`. Style inputs with `border border-surface-gridline rounded px-2 py-1 text-sm`.

3. **Campaign dropdown:** `<select>` with "All campaigns" as default (value=""). Options from `campaigns` prop. On change: `setFilters({ campaign: value || null, adGroup: null })` -- CRITICAL: clear adGroup when campaign changes (per research pitfall #4, prevents stale ad group selection).

4. **Ad group dropdown:** `<select>` with "All ad groups" as default. Options filtered: if a campaign is selected, only show ad groups where `adGroup.campaignId === selectedCampaign`. If no campaign selected, show all. On change: `setFilters({ adGroup: value || null })`.

5. **Clear filters:** A small "Clear" text button (text-brand-grey hover:text-gray-900) that resets all filters to defaults: `setFilters({ range: '30d', from: null, to: null, campaign: null, adGroup: null })`.

Use `lucide-react` icons: `Calendar` for date section, `Filter` for campaign/ad group section.
